Hey everyone! SkyRandom 1.2.0-beta.2 brings a brand-new optional Random Events modifier, some rare new loot, and moves the whole project onto the new Minecraft 26.2 platform.

The headline feature is the new optional Random Events (BETA) modifier — short, game-changing events that fire during a match to keep rounds unpredictable. On top of that there's a rare explosive Sulfur Cube drop and a set of powerful Ultra Rare items. This release also migrates SkyRandom to the Paper/Purpur 26.2 API line, so the supported server versions have changed (see Important below).

New
• Added the optional Random Events (BETA) modifier.

• Added the Low Gravity event — higher jumps and softer falls for 15 seconds.

• Added the Lucky Drops event — only rare loot drops for 10 seconds.

• Added the Resistance event — everyone gets Resistance I for 15 seconds.

• Added the Sudden Hunger event — hunger drains for 15 seconds, then restores.

• Added the Fire Feet event.

• Added a rare explosive Sulfur Cube drop, which spawns near a player with primed TNT already absorbed inside.

• Added new Ultra Rare drops: Ultra Elytra, Ultra Totem, Ultra Horn, Ultra Wind Charge, Ultra Mace, Ultra Enchanted Apple, Ultra Netherite Sword, and Ultra Netherite Axe.

Improvements
• Updated SkyRandom to run on the Paper/Purpur 26.2 API line.

Important
• Supported server software is now Paper 26.2 and Purpur 26.2 (both currently experimental build lines).

• Java 25 is now required.

• Paper/Purpur 1.21.11 and 26.1.x are no longer supported — servers on those versions should stay on the older SkyRandom release line.

This is a feature-packed beta built on a brand-new platform, so Random Events is marked BETA and may change in future updates, and 26.2 itself is still experimental. Feedback and bug reports are always welcome!
