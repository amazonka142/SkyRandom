SkyRandom 1.0.0-beta.4+15

Fourth public beta release of SkyRandom.

Added Sudden Night.

After 5 minutes, the arena switches from day to night for 3 minutes, then returns to day.

Added new chaos effects: Glowing, Weakness, Slowness, Poison, Fire Resistance, Strength, Regeneration, and Resistance.

Effect drops can now use random duration ranges instead of only fixed durations.

Added Golden Apple as a rare gear drop.

Added Enchanted Golden Apple as a very rare ultra_rare drop.

Improved the chaos loot pool to make matches feel more varied and unpredictable.

Fixed build version handling so the plugin version now updates correctly inside the jar.
